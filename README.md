# CRUD-Flask-App
CRUD Application made with Flask Python and SQLite as DB

To use this app:

  1st: Download and Install python to your computer.
        
        https://www.python.org/downloads/
        
  2nd: Create a folder to where you will install this app.
        
        E.g. D:/PythonApp
        
  3rd: Open CMD and CD to your App destination folder 
        
        E.g. 'D:/PythonApp'
  
  4th: Enter following commands.
        
        #Create a virtual environment.
        py -3 -m venv venv
        
        #activate the virtual environment
        venv\Scripts\activate
        
        #Install Flask
        pip install Flask
        
  5th: Copy this repo to your destination folder.
  
  6th: Run server.
        python app.py
        
  7th: Open Browser.
  
        Open your favorite browser and type in:
        127.0.0.1:5000
